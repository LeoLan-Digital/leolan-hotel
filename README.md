# LeoLan Hotel Landingpage

## Feature: Digitaler Concierge (vom Inhaber Leo)

Gast schreibt über WhatsApp oder QR-Code-Chat im Zimmer:
- "Ich brauche eine dünnere Decke, Zimmer 304"
- "Bitte Zimmerservice, Zimmer 207"
- "Extra Handtücher für morgen früh"
- "Taxi für 08:00 Uhr bestellen"

→ KI versteht die Anfrage automatisch
→ Leitet an Rezeption weiter: Zimmer + Anfrage + Uhrzeit
→ Kein Telefon, keine Sprachbarriere, keine Missverständnisse

Dieses Feature unbedingt prominent einbauen — ist das Herzstück der Hotel-Lösung.

## Concierge - Kernbotschaft (von Leo)

"Man braucht keinen Stress im Urlaub. Man ist nicht bereit, eine halbe Stunde zu diskutieren, 
um zu erklären dass man keine normale Decke will, sondern eine leichte."

Lösung:
- Foto hochladen → KI versteht sofort was gemeint ist
- In eigener Sprache schreiben (Deutsch, Türkisch, Arabisch, Russisch...) → KI übersetzt für Rezeption
- Kein Erklären, kein Missverständnis, kein Stress

Headline-Idee: "Zeig einfach drauf. Oder schreib in deiner Sprache. Wir kümmern uns."
